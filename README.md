## Installation

```
yarn
```

```
yarn dev
```

## Config via query params

### Seeds

Custom seeds

```
/?seed=1337
/?seed=tilesgame.com
```

Special seeds

```
/?seed=today
/?seed=yesterday
```

### Grid size

min. 3 and max. 8

```
/?gridSize=3
/?gridSize=8
```

### Tile types

min. 3 and max. 5

```
/?tileTypes=3
/?tileTypes=5
```
